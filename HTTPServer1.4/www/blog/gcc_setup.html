gcc编译环境配置教程

什么是GCC？

GCC 是 GNU Compiler Collection 的缩写，是一个编译器系统，支持多种编程语言。

安装GCC

在不同的操作系统上安装GCC的方法略有不同。

在Linux上安装GCC


sudo apt update
sudo apt install gcc

在macOS上安装GCC


brew install gcc

在Windows上安装GCC

可以使用MinGW或Cygwin来安装GCC。

配置GCC环境变量

安装完成后，需要将GCC添加到环境变量中，以便在命令行中使用。


export PATH=$PATH:/path/to/gcc

更多信息请参考 GCC官方安装指南
