什么是HTML？

HTML 是用来描述网页的一种语言。HTML 指的是超文本标记语言 (Hyper Text Markup Language)。

  • HTML 不是一种编程语言，而是一种标记语言 (markup language)
  • 标记语言是一套标记标签 (markup tag)
  • HTML 使用标记标签来描述网页

更多信息请参考 菜鸟教程 HTML教程

HTML 标签

HTML 标记标签通常被称为 HTML 标签 (HTML tag)。HTML 标签是由尖括号包围的关键词，比如 <html>。

HTML 文档 = 网页

HTML 文档描述网页，HTML 文档包含 HTML 标签和文本。

Web 浏览器的作用是读取 HTML 文档，并以网页的形式显示出它们。

下面是一段简单的示例代码


<html>
<body>

<h1>我的第一个标题</h1>

</body>
</html>
